1.4 Not really, After watching the videos I could piece together the next steps and eventually got a result.

2.1 I used ray marching, which is a teqnique similar to ray tracing, where you for each pixel send out a ray looking for objects in the scene. In ray marching
you describe objects with signed distance functions (sdf's) instead of using traditional vertices. An sdf returns the shortest distance from a point in the room
to the object in question. The ray's go with steps in the scene where each step is the current shortest sdf value, and when this value is below a threshold, it has
then hit something. Then you can use a SmoothUnion function to blend the sdf values to create the floatey effect.
2.2/2.4 Ray marching seems like a good soliution for this, but it is harder to work with than vertex based geometry since you can only use sdf's, so it's good for some effects but really bad for others.
The lack of uniform buffers made it really hard to pass multiple variables to the shader at once, which made me put lots of single uniform values instead which is
really ugly, I could have put the values in a texture instead and then sampled the texture in the shader, but that seemed overkill for only 8 vec3 values. Indexing
arrays was also painfull since they have to be constants, which forced me to do a series of if statements to compare my index value to constants that I actually could
index the array with instead of using the "runtime-index" directly. This could probably be solved in a better way, but felt like too much work for this assignment.
I also have 2 tan operations in the shader which is not the best, I could probably rewrite that into something like a screenHeight/screenWidth constant with trigonometry.
2.3 No I didn't think of other solutions

3.1 I used a SmoothBlending function to make the metaballs look more floatey and normals and a pointlight to make them look 3D, and you can also move the camera with wasd
3.2 see 3.1
3.3 I did prioritization, I wanted the metaballs to be in different colours first and then also blend the colours between them when they were intersecting, but it
